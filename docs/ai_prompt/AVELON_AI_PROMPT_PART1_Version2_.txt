## User Roles & Permissions

### Only Two Roles

| Role | Description |
|------|-------------|
| **Admin** | Platform operators who manage loan plans, review KYC, monitor loans |
| **Borrower** | End users who apply for and manage loans |

**IMPORTANT**:  There are NO external lenders.  Avelon Treasury is the sole lender.

### Borrower Verification States

```
REGISTERED → VERIFIED → CONNECTED → PENDING_KYC → APPROVED/REJECTED
     │           │           │            │              │
     │           │           │            │              └─ Can borrow (if approved)
     │           │           │            └─ Documents submitted, awaiting AI review
     │           │           └─ Wallet connected, no KYC yet
     │           └─ Email verified, no wallet
     └─ Email not verified
```

### Admin Capabilities
- Create/edit/deactivate loan plans
- Review and override AI KYC decisions
- View all users and their credit scores
- Monitor all active loans
- Manually trigger liquidation (emergency)
- Update ETH/PHP price (for demo)
- View platform analytics
- Access treasury management

### Borrower Capabilities
- Register with email, verify email
- Connect and verify wallet (signature-based)
- Upload KYC documents
- View credit score and tier
- Apply for loans (based on tier eligibility)
- Deposit collateral
- Make repayments (partial or full)
- Add collateral to existing loans
- Request loan extension (VIP only, score ≥90)
- View loan history and transactions
